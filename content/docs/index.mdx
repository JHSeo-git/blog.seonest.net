## 로컬에서 실행하기

이 가이드는 Seonest 블로그 프로젝트를 로컬 환경에서 실행하는 방법을 안내합니다.

### 1. 저장소 클론

먼저 GitHub에서 저장소를 클론합니다.

```bash title="bash"
git clone https://github.com/JHSeo-git/blog.seonest.net.git my-blog

cd my-blog
```

### 2. 의존성 설치

`pnpm`을 사용하여 의존성을 설치합니다.

```bash title="bash"
pnpm install
```

### 3. 개발 서버 실행

모든 설정이 완료되면 개발 서버를 실행합니다.

```bash title="bash"
pnpm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하면 블로그를 확인할 수 있습니다.
