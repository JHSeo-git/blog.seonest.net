## Auto-compact: false

- `CLAUDE.md`, `commands`, `sub-agents`, `hooks`을 제대로 설정하면 `auto-compact`을 줄일 수 있음
- `auto-compact`가 발동될 때마다 중요한 컨텍스트가 손실되는 경우가 빈번하게 발생
- 해당 압축 과정을 분석해보니 핵심 세부사항들이 누락
- `auto-compact`를 비활성화하고, 세션별로 컨텍스트를 더 잘 관리하는 방식으로 전환

## Verbose output: true

- 상세 로그로 디버깅/모니터링 용이

## Default permission mode: Plan Mode

- Claude와 의견을 주고받으며 만족스러운 계획 수립 가능
- 계획 완성 후 편집 자동 승인 모드로 전환하면 Claude가 한 번에 작업 완료
- PR 작성 같은 체계적인 작업에 특히 유용
- 좋은 계획을 세우는 것이 성공적인 결과의 핵심

## Model: opus

- Sonnet보다 크고 느리지만 더 정교한 추론 가능
- 더 적은 횟수로 원하는 결과를 얻을 확률이 높음
- 도구 활용 능력이 뛰어남
- 복잡한 코딩 작업에서 결과적으로 더 효율적일 수 있음
- 재시도 횟수가 줄어들어 시간 절약 효과
